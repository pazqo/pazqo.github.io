# Graeco-Latin square

This is a pretty rare Sudoku, as it's extremely restricted. Let's take a look at more generic problem of **Graeco-Latin square**, or **orthogonal Latin squares**.

Given a *nxn* square and an alphabeth with *n* symbols, we can form a latin square if each rown and each column contain a set of the symbols of the alphabeth. In the Sudoku, we also require that this is true for each region.

Given a *nxn* square and two alphabets (in our case numbers and colors) with *n* symbols, we want to arrange a pair of symbols in each cell such that when restricting to a single alphabeth we get a latin square, and no pair is repeated.
